ભક્તિ સાધનો भक्ति साधन Spiritual Tools

કંટ્રોલ બોક્સની ઉપર આપેલ આઇકોનથી તમે ઘંટડી 🔔, શંખ 🐚, ફૂલ 🌸 અને દીવો 🪔 પ્રગટાવી શકો છો. कंट्रोल बॉक्स के ऊपर दिए गए आइकन से आप घंटी 🔔, शंख 🐚, फूल 🌸 और दीया य जला सकते हैं। Use festive icons above controls to ring the Bell 🔔, blow the Shankh 🐚, offer Flowers 🌸, or light the Lamp 🪔.
🎵

ભજન અને મંત્ર भजन और मंत्र Audios

કોઈ પણ લિસ્ટમાંથી ભજન પર ક્લિક કરો. તમે નીચેના પ્લેયરથી તેને ચાલુ કે બંધ કરી શકો છો. किसी भी लिस्ट से भजन पर क्लिक करें। आप नीचे दिए गए प्लेयर से इसे कंट्रोल कर सकते हैं। Select any title to play. Control playback via the floating bottom bar.
🌍

ભાષા અને રીમાઇન્ડર भाषा और रिमाइंडर Preferences

મેનુમાં આપેલ 'ગ્લોબ' આઇકોનથી ભાષા અને 'ઘડિયાળ' આઇકોનથી રીમાઇન્ડર સેટ કરો. मेनू में 'ग्लोब' आइकन से भाषा और 'घड़ी' आइकन से रिमाइंडर सेट करें। Switch language via the Globe icon and set custom prayer reminders via the Clock icon.
